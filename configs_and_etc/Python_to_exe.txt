через cmd установить
pip install pyinstaller
если что-то пойдёт не так, то возможно нужно
pip install wheel
и потом опять установить pyinstaller

в папке с программой в проводнике нажать справа от названия папки
и туда написать cmd и нажать Enter (запуск cmd в директории папки)

в открывшейся командной строке
pyinstaller -F -w -i "Путь к иконке !обяз .ico" название.py
результат - в папке dist
-F - в папке dist будет единственный (полный) ехе файл - лучше всегда писать
-w - не будет показываться командная строка при запуске - иногда нужно
-i "Путь" - иконка - по желанию

Антивирус может блокировать и удалять файлы
pyinstaller`а и созданные ехе программы
